ikit

  • Essentially a bunch of JQuery plugins
  • Some cases applied automatically based on css classes, and helper methods.
  • Handles awkward internationalization like date/time via moment.js. Also can access message properties(IKitMessageFactory).
  • webkit-oriented, so works well on iOS/Droid only
  • Improve UX in untrobursibe way, unlike sencha touch or jquery mobile.

Using ikit

Two stylesheets
  • - ikit.css The first one is required and defines basic layouts and structures.
  • - ikit-skin.css - can be completely replaced or overridden with custom styles.
